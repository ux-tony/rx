# RX Architect MVP

Минималистичный MVP веб-сайта-портфолио для архитектурной студии. Проект представляет собой веб-приложение для презентации архитектора, кейсов и визуального подхода студии в лаконичном журнальном формате.

## О проекте

Сайт собран как frontend MVP с адаптивным интерфейсом, минималистичной визуальной системой и структурой, подготовленной для дальнейшего подключения CMS, реальных кейсов и контентного управления.

## Стек

- Next.js 15
- React 19
- TypeScript
- Mantine 7

## Запуск

```bash
npm install
npm run dev
```
