# RX Architect MVP

Минималистичный MVP сайта-портфолио архитектора на `Next.js + TypeScript + Mantine`.

## Что реализовано

- hero-экран с крупным визуалом и акцентом на архитектурную подачу
- адаптивная сетка проектов в эстетике архитектурного портфолио
- блок подхода и направлений работы
- FAQ на Mantine с кастомной стилизацией
- контактный CTA-блок
- mock-данные для проектов и контента

## Стек

- Next.js 15
- React 19
- TypeScript
- Mantine 7

## Запуск

```bash
npm install
npm run dev
```
